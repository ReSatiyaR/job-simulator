آموزش رفع باگ

باگ یک خطا در کد است که باعث می‌شود برنامه به‌درستی کار نکند. برای رفع باگ، باید خطای موجود در کد را پیدا کرده و آن را اصلاح کنید.

مثال: در کد زیر، عملگر اشتباه استفاده شده است. به جای تفریق، باید جمع انجام شود.

function add(a, b) {
    return a - b; // باگ: باید جمع شود، نه تفریق!
}

رفع باگ

کد زیر دارای یک باگ است. آن را پیدا کرده و رفع کنید:

function add(a, b) {
    return a - b; // باگ: باید جمع شود، نه تفریق!
}
